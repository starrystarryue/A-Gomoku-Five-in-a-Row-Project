# A-Gomoku-Five-in-a-Row-Project
the final project for the course Introduction to Computation (A), implemented in C++

A Gomoku (Five in a Row) game based on C++ has been developed, featuring a minimalist yet aesthetically pleasing UI design with background music, emphasizing user experience. The project supports both human-computer and two-player modes, and fully implements core functionalities such as forbidden move rules.
